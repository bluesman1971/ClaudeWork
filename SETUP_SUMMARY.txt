STEP 1: Install Python Packages
------
Open Terminal/Command Prompt in trip-guide-app folder:

  python -m venv venv
  source venv/bin/activate              # Mac/Linux
  venv\Scripts\activate                 # Windows
  pip install -r requirements.txt


STEP 2: Set Up API Key
------
Copy .env.example to .env:

  cp .env.example .env                  # Mac/Linux
  copy .env.example .env                # Windows

Edit .env and add your Anthropic API key:

  ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxx

Get API key from: https://console.anthropic.com/api-keys


STEP 3: Run Everything
------
Terminal 1 (Backend):
  python app.py

  Expected output:
  "Backend running on: http://localhost:5001"

Terminal 2 (Frontend):
  python -m http.server 5000

  Expected output:
  "Serving HTTP on 0.0.0.0 port 5000"

Browser:
  Open: http://localhost:5000

  You should see the Trip Master form!


================================================================================
                            WHAT HAPPENS NEXT
================================================================================

When you fill out the form and submit:

1. Frontend sends user preferences to http://localhost:5001/generate
2. Flask backend receives the request
3. Backend calls Anthropic API 3 times (for each scout):
   - Photo Scout: Generates photography locations
   - Restaurant Scout: Generates restaurant recommendations
   - Attraction Scout: Generates attractions & things to do
4. Backend orchestrates all three responses
5. Backend generates unified HTML with consistent styling
6. Backend generates PDF using weasyprint
7. Results sent back to browser as HTML + PDF
8. User sees preview and can download PDF


================================================================================

  - Photo Scout API call: 30-40 seconds

================================================================================
                          TROUBLESHOOTING
================================================================================

ERROR: Connection refused (localhost:5001)
  FIX: Make sure Terminal 1 has "python app.py" running

ERROR: CORS error / Failed to fetch
  FIX: Verify Flask is running on port 5001, frontend on 5000

ERROR: "Cannot GET /" on http://localhost:5000
  FIX: Make sure Terminal 2 has "python -m http.server 5000" running

ERROR: API key error
  FIX: Check .env has real key starting with "sk-ant-"
      Restart Flask after changing .env

ERROR: "Cannot find module"
  FIX: Make sure venv is activated and pip install ran
       Check: pip list (should show Flask, anthropic, weasyprint)

ERROR: Very slow (takes 5+ minutes)
  FIX: First request is slow - this is normal
       Subsequent requests will be faster
